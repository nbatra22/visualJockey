How I Built This

My first step for this project was implementing the audio analysis engine. Here, I used the Web Audio API to extract frequency, bass, rhythm and musicality. For the accompanying visual layer, I started with initial p5.js sketches in the "Waveform" environment. Eventually I refined these sketches and also added high-performance 3D rendering using Three.js and Simplex-noise for the "Particles" and "3D Blob" environments.

My next step was the AI layer. Here I utilized Google AI Studio to write a system prompt for the Gemini 2.5 Flash Lite Model API. I use Gemini as basically the director: analyzing the user’s prompt for sentiment and mood and manipulating the mathematical parameters of each animation environment. To view the raw animations without AI manipulation, run "START VISUALIZING" without a user prompt.

My initial prototype asked Gemini to generate an entire, injectable rendering script from scratch. However, the results lacked both soul and reliability causing me to scrap this idea. I also explored Google's Imagen tool for vector-based images and animations, but found the outputs felt tacky and disconnected from the audio.
